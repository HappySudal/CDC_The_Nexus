# PPT 다이어그램 & 인포그래픽 레퍼런스 | Diagram & Infographic Resources

프레젠테이션의 **핵심인 다이어그램과 인포그래픽**을 설계하기 위한 도구, 영감, 그리고 베스트 프랙티스입니다.  
Essential resources for creating compelling diagrams and infographics in presentations.

## 🛠️ 다이어그램 & 인포그래픽 제작 도구 | Creation Tools

### 1. **Infogram** (데이터 중심)
- **URL**: https://infogram.com/
- **주요 기능**:
  - 데이터 시각화 (차트, 그래프, 맵)
  - 상호작용형 요소 (클릭 가능 맵, 애니메이션)
  - 템플릿 (1000+ 디자인)
  - 실시간 데이터 연동 (Google Sheets 등)
- **PPT 활용법**:
  - 차트/그래프 제작 → PNG 다운로드 → PPT 삽입
  - 인포그래픽 템플릿 → 데이터 입력 → 이미지 내보내기
  - 컬러 커스터마이징 (design-brief의 팔레트 맞춤)
- **추천 시나리오**: 데이터 시각화, 통계 자료

### 2. **Piktochart** (초보자 친화)
- **URL**: https://piktochart.com/
- **주요 기능**:
  - 드래그앤드롭 인포그래픽 제작
  - 아이콘/일러스트 라이브러리
  - 템플릿 (인포그래픽, 포스터, 보고서)
  - 브랜드 색상 팔레트 저장 기능
- **PPT 활용법**:
  - design-brief의 색상 입력 → 템플릿 선택 → 커스터마이징
  - 아이콘 사용 → 슬라이드 다이어그램에 일관성
  - 내보내기 (SVG/PNG) → 고해상도 유지
- **추천 시나리오**: 프로세스 플로우, 통계 인포그래픽

### 3. **Venngage** (전문가 수준)
- **URL**: https://venngage.com/
- **주요 기능**:
  - 프로 레벨 인포그래픽 템플릿
  - 브랜드 키트 (색상, 폰트, 로고)
  - 접근성 검사 (대비도, 판독성)
  - 차트/데이터 시각화 고급 옵션
- **PPT 활용법**:
  - 브랜드 키트에 design-brief 색상 등록
  - 프로 템플릿 활용 (기업 보고서, 데이터 분석)
  - 접근성 검증 후 내보내기
- **추천 시나리오**: 고급 인포그래픽, 보고서 수준

### 4. **Miro** (협업 & 고급 다이어그램)
- **URL**: https://miro.com/templates/
- **주요 기능**:
  - 무한 캔버스 (대형 다이어그램 가능)
  - 협업 실시간 작업
  - 템플릿 (프로세스 맵, 저니 맵, 마인드맵 등)
  - 형태, 연결선, 아이콘 라이브러리
- **PPT 활용법**:
  - 복잡한 프로세스 다이어그램 설계
  - 팀과 협력 → 최종 이미지 캡처
  - PPT에 삽입 (PNG 또는 SVG)
- **추천 시나리오**: 고급 다이어그램, 팀 협업

### 5. **Figma Presentations**
- **URL**: https://www.figma.com/community/presentations
- **주요 기능**:
  - 커뮤니티 제작 프레젠테이션 템플릿
  - 디자인 시스템 기반 (색상, 타이포 일관성)
  - 실시간 협업
  - 프리뷰로 스타일 확인 가능
- **PPT 활용법**:
  - design-archetypes와 유사한 스타일 찾기
  - 레이아웃/다이어그램 패턴 학습
  - SVG 또는 이미지로 내보내기 → PPT 참고
- **추천 시나리오**: 디자인 시스템 학습, 패턴 참고

### 6. **Canva** (빠른 프로토타이핑)
- **URL**: https://www.canva.com/
- **주요 기능**:
  - 160만+ 템플릿
  - 드래그앤드롭 인터페이스
  - 스톡 사진, 아이콘, 도형
  - 팀 협업 (브랜드 키트 공유)
- **PPT 활용법**:
  - 프레젠테이션 카테고리에서 다이어그램 탐색
  - design-brief 색상으로 커스터마이징
  - PNG/PDF 다운로드 → PPT에 삽입
- **추천 시나리오**: 빠른 프로토타이핑, 템플릿 기반

### 7. **Visme** (올인원 솔루션)
- **URL**: https://www.visme.co/
- **주요 기능**:
  - 인포그래픽 + 프레젠테이션 + 비디오
  - 애니메이션 지원
  - 고급 차트/다이어그램
  - 브랜드 일관성 도구
- **PPT 활용법**:
  - 애니메이션 다이어그램 제작 → 비디오 녹화
  - 또는 정적 이미지로 내보내기
- **추천 시나리오**: 애니메이션이 필요한 복잡 다이어그램

### 8. **Behance / Dribbble** (영감 갤러리)
- **URL**: https://www.behance.net/, https://dribbble.com/
- **주요 기능**:
  - 전문 디자이너 인포그래픽 갤러리
  - 다양한 스타일 & 산업별 예시
  - 색상 팔레트 분석
  - 디자이너 포트폴리오 연결
- **PPT 활용법**:
  - 산업별 다이어그램 스타일 조사
  - 색상 팔레트 추출 (Adobe Color)
  - 구조/계층 패턴 학습
- **추천 시나리오**: 스타일 리서치, 패턴 분석

## 🎯 다이어그램 설계 가이드 | Diagram Design Principles

### 1. 색상 일관성
- **규칙**: design-brief의 팔레트만 사용
- **패턴**:
  - Primary: 주요 요소 (프로세스 단계, 주제)
  - Secondary: 보조 요소 (데이터, 라벨)
  - Accent: 강조 (중요 데이터, CTA)
- **예시**: Process Flow → Primary(단계박스) → Secondary(화살표) → Accent(중요단계)

### 2. 타이포그래피 일관성
- **규칙**: Pretendard (제목) + Noto Sans KR Medium (본문)
- **크기**:
  - 제목: 20-24pt
  - 라벨: 14-16pt
  - 설명: 12-14pt

### 3. 여백 및 정렬
- **원칙**: 
  - 최소 0.3-0.5" 요소 간 간격
  - 명확한 정렬 (좌측, 중앙, 우측 일관)
  - 계층 구조 표현 (크기, 위치, 색상)

### 4. 아이콘 & 도형
- **일관성**: 같은 스타일 사용 (모두 outline 또는 filled)
- **크기**: 비례적 (중요도에 따라 조정)
- **색상**: design-brief 팔레트 준수

### 5. 애니메이션 (선택사항)
- **추천**: 
  - 프로세스 플로우 → 단계별 나타내기
  - 데이터 차트 → 증가 애니메이션
- **주의**: 과도한 애니메이션 피하기 (전문성 손상)

## 📚 다이어그램 패턴 라이브러리 | Common Diagram Patterns

### 패턴 1: 프로세스 플로우 (Process Flow)
```
[시작] → [단계1] → [단계2] → [의사결정] 
                                  ↓ Yes/No
                              [결과] → [끝]
```
**색상**: Primary(단계) → Accent(의사결정)
**도구**: Miro, Venngage

### 패턴 2: 2x2 매트릭스 (2x2 Matrix)
```
        High Impact
            │
Low Cost    │    High Cost
    ─────────┼─────────
            │
        Low Impact
```
**색상**: 각 사분면별 다른 Secondary 색상
**도구**: Figma, Miro

### 패턴 3: 계층 구조 (Hierarchy)
```
        Level 1
       /  │  \
    L2-A L2-B L2-C
    / \      / \
  L3  L3   L3  L3
```
**색상**: 깊이에 따라 Color 그래디언션
**도구**: Venngage, Miro

### 패턴 4: 타임라인 (Timeline)
```
2024 Q1 ─── 2024 Q2 ─── 2024 Q3 ─── 2024 Q4
  │           │           │           │
Milestone   Milestone   Milestone   Milestone
```
**색상**: Accent로 주요 마일스톤 강조
**도구**: Venngage, Canva

### 패턴 5: 비교 칼럼 (Comparison Columns)
```
Option A          Option B          Option C
├─ Feature 1   ├─ Feature 1   ├─ Feature 1
├─ Feature 2   ├─ Feature 2   ├─ Feature 2
└─ Feature 3   └─ Feature 3   └─ Feature 3
```
**색상**: Option별 다른 Secondary 색상
**도구**: Infogram, Venngage

## 💡 다이어그램 제작 워크플로우 | Creation Workflow

### Step 1: 선택 (Selection)
```
다이어그램 유형 결정
   ↓
도구 선택 (위 추천 테이블 참고)
   ↓
템플릿 검색 또는 백지에서 시작
```

### Step 2: 설계 (Design)
```
design-brief 색상 팔레트 준비
   ↓
Pretendard + Noto Sans KR Medium 설정
   ↓
프로토타입 작성
```

### Step 3: 정제 (Refinement)
```
색상 일관성 확인
   ↓
여백/정렬 검증
   ↓
아이콘/도형 스타일 확인
```

### Step 4: 내보내기 (Export)
```
SVG 또는 PNG로 내보내기
   ↓
고해상도 확인 (최소 150 DPI)
   ↓
PPT에 삽입
```

## 🔗 Behance/Dribbble 다이어그램 영감 검색 팁 | Inspiration Search Tips

### 검색 키워드
- "infographic" + 산업 (예: "fintech infographic")
- "diagram" + 유형 (예: "process diagram")
- "flow chart"
- "data visualization"
- "timeline"
- "hierarchy"

### 필터 활용
1. **Dribbble**: Tags → "infographic", "diagram", "visualization"
2. **Behance**: Industry 필터 → 자신의 산업 선택
3. 상위 평점순 (커뮤니티 검증)

   ↓
Adobe Color 이미지 업로드
