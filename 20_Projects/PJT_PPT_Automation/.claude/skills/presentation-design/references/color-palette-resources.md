프레젠테이션 디자인의 **핵심 요소인 색상 조합**을 개발하기 위한 도구 및 영감 소스입니다.  
Essential tools and inspiration sources for developing presentation color palettes.

## 🎨 색상 생성 및 탐색 도구 | Color Generation Tools

### 1. **Adobe Color** (Adobe.com의 고급 기능)
- **URL**: https://color.adobe.com/
- **주요 기능**:
  - AI 트렌드 분석 (최신 디자인 트렌드 색상)
  - 이미지에서 자동 색상 추출
  - 색상 규칙 기반 생성 (Complementary, Triadic, Analogous 등)
  - 접근성 검사 (대비도 WCAG 검증)
- **PPT 활용법**:
  - 에이전시 포트폴리오 스크린샷 업로드 → 색상 추출
  - Hex 코드 직접 복사 → PowerPoint 컬러 팔레트 입력
  - 트렌드 색상 확인 → 현대성 확보
- **추천 시나리오**: 고급 브랜드 가이드 필요 시

### 2. **Coolors.co** (빠른 생성 도구)
- **URL**: https://coolors.co/
- **주요 기능**:
  - 스페이스바 한 번으로 5색 팔레트 자동 생성
  - Lock 기능 (마음에 드는 색상 고정 후 나머지만 재생성)
  - 색상 정보 검색 (이름, 유사 색상, 시즌별 색상)
  - 플러그인 (Figma, Chrome 등)
- **PPT 활용법**:
  - 초기 팔레트 스캐핑: 5색 생성 → Lock → 재생성 반복
  - 색상 이름 확인 (RGB, HSL, HEX 동시 표시)
  - Export 기능으로 이미지/코드 저장
- **추천 시나리오**: 빠른 프로토타이핑, 초기 아이디어 단계

### 3. **Color Hunt** (커뮤니티 팔레트)
- **URL**: https://colorhunt.co/
- **주요 기능**:
  - 디자이너 커뮤니티가 만든 4색 팔레트 (수만 개)
  - 인기도 순, 최신순, 감정 기반 필터 (Happy, Calming, Vivid 등)
  - 각 팔레트별 사용 도구 표시
  - 팔레트 다운로드 (이미지 또는 코드)
- **PPT 활용법**:
  - 감정 필터로 브랜드 톤 매칭 (Calming → 보수, Happy → 에너지)
  - "최신" 필터로 트렌드 색상 확인
  - 4색 팔레트 직접 PPT에 적용 가능
- **추천 시나리오**: 커뮤니티 검증된 색상, 트렌드 리서치

### 4. **Behance Color Palettes**
- **URL**: https://www.behance.net/search/projects/color%20palette
- **주요 기능**:
  - 전문 디자이너의 실제 프로젝트 색상 조합
  - 필터 (산업, 스타일, 감정)
  - 프로젝트 소개글에서 색상 사용 배경 확인 가능
  - 디자이너 포트폴리오로 직접 이동
- **PPT 활용법**:
  - 업계별 색상 연구 (예: "fintech" 검색 → 금융 기술 색상 트렌드)
  - "왜 이 색상을 선택했는가" 읽기 → 색상 철학 학습
  - 실제 사례 분석 (단순 팔레트가 아닌 적용 사례)
- **추천 시나리오**: 깊이 있는 색상 연구, 산업별 분석

### 5. **Dribbble Color Inspiration**
- **URL**: https://dribbble.com/tags/color-palette-inspiration
- **주요 기능**:
  - 최신 디자인 트렌드 색상 (거의 실시간)
  - 디자이너 평점 기반 정렬
  - UI/Print/Brand 등 카테고리별 필터
  - 색상 값 복사 기능 (일부 디자인)
- **PPT 활용법**:
  - 현재 트렌드 색상 파악
  - 같은 카테고리(Presentation 등) 내 색상 패턴 분석
  - 고평점 디자인만 참고 (커뮤니티 신뢰도 높음)
- **추천 시나리오**: 최신 트렌드, 업계별 현재 기준

## 🎯 색상 팔레트 개발 프로세스 | Color Palette Development Process

### Step 1: 영감 수집 (Discovery)
```
에이전시 포트폴리오 탐색
   ↓
스크린샷 → Adobe Color (이미지 업로드)
   ↓
Hex 코드 3-5개 추출
```

### Step 2: 후보 생성 (Ideation)
```
Coolors.co: 스페이스바로 빠른 생성
   ↓
Lock + 재생성 반복 (5-10개 팔레트)
   ↓
Color Hunt / Dribbble: 커뮤니티 검증
```

### Step 3: 검증 (Validation)
```
각 팔레트 프린트아웃
   ↓
design-archetypes.md 스타일과 비교
   ↓
WCAG 대비도 확인 (Adobe Color)
```

### Step 4: 최종화 (Finalization)
```
Top 3 선택
   ↓
presentation-design 스킬에서 설명
   ↓
design-brief.md에 Hex 코드 기록
```

## 💡 색상 선택 가이드라인 | Color Selection Guidelines

### 주요 색상 역할 분담 (PPT 관점)

| 역할 | 수량 | 용도 | 예시 |
|------|------|------|------|
| **Primary | 1개 | 제목, 주요 강조, 헤더 배경 | Navy #1A2B4C (Global Insight) |
| **Secondary** | 1-2개 | 섹션 강조, 차트, 보조 요소 | Cobalt #2E5BFF |
| **Accent** | 1개 | 중요 데이터, CTA, 하이라이트 | Red #E4002B |
| **Neutral** | 2개 | 텍스트, 배경, 분할선 | Gray #F5F5F5, Black #000 |

### 팔레트 조합 유형

#### 타입 A: 보수적 (Corporate)
- 주색: Navy / Charcoal / Deep Green
- 보조색: Light Gray / Cream
- 강조색: Gold / Teal
- 추천: McKinsey Style (Global Insight)

#### 타입 B: 현대적 (Modern)
- 주색: Deep Indigo / Charcoal
- 보조색: Light Gray / White
- 강조색: Electric Cyan / Neon
- 추천: Gartner Style (Future Signals)

#### 타입 C: 창의적 (Creative)
- 주색: Black / Deep Purple
- 보조색: Off-White / Cream
- 강조색: Saturated accent (Yellow/Pink/Orange)
- 추천: Pentagram Style (Systemic Identity)

#### 타입 D: 친근한 (Approachable)
- 주색: Warm Gray / Soft Blue
- 보조색: Light Pastel
- 강조색: Warm Orange / Coral
- 추천: IDEO Style (Connected Systems)

## 🔍 색상 대비도 확인 | Contrast Checker

### WCAG 접근성 기준
- **AAA (최고 수준)**: 7:1 이상 (권장)
- **AA (표준)**: 4.5:1 이상 (최소 기준)
- **실패**: 4.5:1 미만

### 확인 방법
1. Adobe Color → "Accessibility" 탭
2. 텍스트 색상 vs 배경 색상 선택
3. 대비도 비율 확인
4. "WCAG AA" 또는 "AAA" 달성 확인

## 📊 PPT 색상 팔레트 예시 | Sample Palettes

### 예시 1: McKinsey Style
```
Primary:    #1A2B4C (Navy)
Secondary:  #F5F5F5 (Light Gray)
Accent:     #2E5BFF (Cobalt)
Text:       #000000 (Black) / #F5F5F5 (White)
```

### 예시 2: IDEO Style
```
Primary:    #FF6B6B (Coral)
Secondary:  #4ECDC4 (Teal)
Accent:     #FFE66D (Yellow)
Text:       #FFFFFF (White) / #2D2D2D (Dark Gray)
Background: #FFFFFF (White)
```

### 예시 3: Pentagram Style
```
Primary:    #F4FF00 (Acid Yellow) OR #FF3CAC (Hot Pink)
Secondary:  #000000 (Black)
Text:       #FFFFFF (White) / #000000 (Black)
Background: #FFFFFF (White) / #000000 (Black)
```
