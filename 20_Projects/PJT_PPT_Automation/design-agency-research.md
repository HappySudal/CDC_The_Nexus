  - MasterCard, MIT, Windows 등 아이콘적 프로젝트

- **적용**: 로고, 아이덴티티, 타이포그래피, 그리드 시스템 영감

- **공식 웹사이트**: https://www.ideo.com/
- **포트폴리오 페이지**: https://www.ideo.com/work
- **특징**:
  - 휴먼 중심 디자인 (Human-Centered Design)

- **공식 웹사이트**: https://www.ogilvy.com/
- **디자인 포트폴리오**: https://www.ogilvy.com/work/design

  - 창의적 캠페인 → 비주얼 아이덴티티
- **적용**: 브랜드 스토리 시각화, 이모셔널 디자인 요소

## 2. 색상 조합 레퍼런스

### A. 색상 팔레트 생성 & 탐색 도구

#### **Adobe Color (Adobe Express Color)**
- **URL**: https://color.adobe.com/
- **탐색 페이지**: https://color.adobe.com/explore
- **특징**:
  - Adobe의 공식 색상 팔레트 라이브러리
  - AI 기반 트렌딩 색상 분석
  - 이미지에서 색상 추출 가능 (Color Capture)
  - 여러 색상 이론 모드 (Analogous, Monochromatic, Triadic, Complementary 등)
  - 브랜딩, 소셜미디어, 웹, 모바일 등 카테고리별 팔레트
  - 다운로드 가능 (ASE, SVG, JSON 형식)
- **PPT 활용**:
  - Pantone + Hex 코드로 정확한 색상 지정
  - 진영 간 색상 일관성 관리
  - 트렌드 색상 연간 예측 (Pantone Fashion Color Report 참고)

#### **Coolors.co**
- **URL**: https://coolors.co/
- **트렌딩 팔레트**: https://coolors.co/palettes/trending
- **특징**:
  - 초고속 팔레트 생성 (스페이스바 누르기)
  - 수백만 개의 인기 팔레트 탐색
  - 색상별 필터링 & 정렬 (Popularity, Rating, Date 등)
  - Figma, Adobe 플러그인 지원
  - iOS/Android 앱 제공
  - 커뮤니티 팔레트 공유
- **PPT 활용**:
  - 빠른 팔레트 브레인스토밍 (스페이스바 → 즉시 5색 제안)
  - 트렌딩 조합 참고
  - 색상 간 명도/채도 시뮬레이션

#### **Color Hunt**
- **URL**: https://colorhunt.co/
- **특징**:
  - 아티스트/디자이너 생성 팔레트 커뮤니티
  - 매일 새로운 조합 업로드 (디자이너 기여)
  - "Trendy / New / Random" 탭
  - 팔레트당 4색 (프레젠테이션 친화적)
  - 색상 코드 복사 용이 (HEX, RGB, HSL)
  - 팔레트 평점 & 다운로드 수 표시
- **PPT 활용**:
  - 감각적, 트렌디한 색상 조합 영감
  - 4색 팔레트는 PPT 슬라이드 마스터에 바로 적용 가능
  - 배경색 + 강조색 + 텍스트색 + 악센트색 구성

## 6. 핵심 활용 전략

### 색상 선택 프로세스

```
Step 1: 목적 & 브랜드 정의
├─ 타깃 오디언스 분석
├─ 이모션 톤 정의 (Professional / Creative / Energetic 등)
└─ 경쟁사 색상 분석 (Differentiation)

Step 2: 팔레트 탐색
├─ Adobe Color Trends → 연도별 트렌드
├─ Coolors → 빠른 생성 (스페이스바)
└─ Color Hunt → 감각적 조합

Step 3: 검증 & 정제
├─ Behance/Dribbble → 실제 프로젝트 적용 확인
├─ 명도/채도 대비 검증 (접근성)
└─ 3-5색 팔레트로 정리 (Primary + Secondary + Accent)

Step 4: 시스템화
├─ Hex 코드 기록 (예: #1F77B4, #FF7F0E)
├─ 색상명 정의 (예: Primary Blue, Accent Orange)
└─ PPT 슬라이드 마스터에 저장
```

### 다이어그램 설계 프로세스

```
복잡도 판단
├─ 간단 (박스 3-5개): PowerPoint 도형 + 화살표로 충분
├─ 중간 (프로세스 흐름): Piktochart / Infogram 템플릿
└─ 고급 (상호작용 필요): Miro / Figma에서 설계 → PNG 내보내기

콘텐츠 구조화
├─ 상향식: 기초 데이터 → 요약 → 결론
├─ 좌→우: 순차적 프로세스
└─ 계층형: 중앙 개념 → 방사형 세부사항

스타일 선택
├─ 미니멀 (Pentagram 참고): 화이트스페이스 + 그리드
├─ 창의적 (Sagmeister & Walsh 참고): 비정형 구성 + 타이포그래피
└─ 데이터 중심 (Infogram 참고): 숫자 + 차트 강조
```

### PPT 슬라이드 마스터 설정

```
필수 요소:
1. 색상 팔레트 (3-5색) → 모든 슬라이드 일관성
2. 타이포그래피 (제목/본문/강조) → 디자인 가이드
3. 레이아웃 마스터 (제목 슬라이드, 콘텐츠, 차트 등)
4. 로고 + 브랜드 요소 위치
5. 다이어그램 템플릿 (자주 쓰는 형태 미리 제작)

추천 워크플로우:
├─ Figma에서 마스터 설계 (협업)
├─ 최종 PNG/PDF로 내보내기
└─ PowerPoint에서 슬라이드 마스터로 적용
```

### 에이전시별 영감 활용

| 에이전시 | 전문성 | PPT에 적용 | 참고 요소 |
|:---:|:---|:---|:---|
| **Pentagram** | 미니멀 + 그리드 | 정돈된 레이아웃, 타이포그래피 | 마스터 그리드 설계 |
| **IDEO** | 휴먼중심 + 경험 | 사용자 플로우, 스토리보드 | 단계별 시각화 |
| **MetaDesign** | 규모확장성 | 대규모 시스템 시각화 | 모듈형 아이콘 세트 |
| **Sagmeister & Walsh** | 창의적 + 구성 | 비정형 레이아웃, 타이포 강조 | 대담한 색상 대비 |
| **Ogilvy** | 스토리텔링 | 감정적 메시지 전달 | 이미지 + 텍스트 조화 |
