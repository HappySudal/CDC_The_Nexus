## 🎯 research-agent 검색 팁 | Research Agent Tips

### 주제별 추천 사이트 조합 | Recommended Site Combinations

| 주제 | 추천 사이트 | 검색 우선순위 |
|---|---|---|
| **글로벌 시장 규모** | Markets and Markets → Statista → Grand View Research | 1순위: M&M, 2순위: Statista |
| **기술 트렌드** | Gartner → a16z → IEEE Spectrum | 1순위: Gartner, 2순위: a16z |
| **경제 지표** | IMF Data → OECD Data → World Bank | 1순위: OECD, 2순위: World Bank |
| **암호화폐 시장** | Chainalysis → a16z Crypto → CoinMarketCap | 1순위: Chainalysis, 2순위: a16z |
| **경영 전략** | McKinsey → BCG → Bain Insights | 1순위: McKinsey, 2순위: BCG |
| **국내 시장** | Naver DataLab → KOSIS → Mintel Korea | 1순위: Naver, 2순위: KOSIS |

### 신뢰도 높은 자료 찾기 | Finding High-Authority Sources

1. **컨설팅 회사 보고서** (최고 신뢰도)
   - McKinsey, BCG, Bain, Kearney → PDF 다운로드 가능
   - 권장: site:mckinsey.com filetype:pdf "market" OR "analysis"

2. **정부/국제기구 통계** (높은 신뢰도)
   - OECD, World Bank, IMF, UN → 공식 데이터
   - 권장: site:data.oecd.org OR site:data.worldbank.org

3. **시장조사 기관** (높은 신뢰도)
   - Gartner, Statista, IBISWorld → 권위 있는 분석
   - 권장: site:gartner.com OR site:statista.com filetype:pdf

4. **언론사 심층보도** (중상 신뢰도)
   - NYT, Reuters, BBC, WSJ → 검증된 저널리즘
   - 권장: site:reuters.com "analysis" OR "report" 2024

5. **VC/투자사 인사이트** (중상 신뢰도)
   - a16z, ARK → 산업 전문가 관점
   - 권장: site:a16z.com OR site:ark-invest.com
