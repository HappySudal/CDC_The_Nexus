## 🔗 포트폴리오 탐색 팁 | Portfolio Exploration Tips

### 에이전시 포트폴리오 방문 체크리스트
1. **홈페이지 → Work/Portfolio 섹션 진입**
2. **최근 프로젝트 3-5개 열어보기**
3. **색상 팔레트 스크린샷 저장** (나중에 Coolors에 입력)
4. **레이아웃/다이어그램 패턴 분석**
5. **타이포그래피 & 여백 방식 메모**
6. **자신의 프레젠테이션 톤과 비교**

### 색상 팔레트 추출 방법
```
에이전시 포트폴리오 →  스크린샷 →  
Adobe Color (이미지 업로드) →  
Hex 코드 추출 →  
Coolors에 저장
```
